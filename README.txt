
Files Required
==============
Create folder GenTrack

The following files need to reside in the folder GenTrack :-

	example.txt
	fail.txt
	input.txt
	large_input.txt
	medium_input.txt
	small.txt
	sparse_medium.txt
	findRoute.java

Directions to Compile and Execute
=================================

Change default to GenTrack compile from command prompt as follows :-

	 C:\GenTrack>javac -cp ../ findRoute.java

If compile is successful then execute as follows :-

	C:\GenTrack>java -cp ../ GenTrack.findRoute

All the above txt files need to be present in the local folder, if any are missing then
an error message will be returned and the next txt file will be processed.
